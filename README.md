# JukuMate — 個人塾長先生向け AI 業務支援 LP（架空ポートフォリオ作品）

> 本サイトは こすらぼ（Co-Thrive Labs）の **架空ポートフォリオ作品** です。実サービスではありません。
> 掲載されている数値・導入事例・サービス内容はすべて、ポートフォリオ訴求のための想定値・架空のものを含みます。

---

## 概要

**JukuMate** は、中学受験・高校受験が混在する小規模な個人塾の塾長先生・オーナー先生に向けて、
日々の業務のなかで「気を遣うけれど時間がかかる仕事」を AI で軽くするための業務支援サービスを想定したランディングページです。

塾長先生が抱えがちな次の悩みに、AI で寄り添うかたちを提案しています。

- 保護者向けレポートの下書きに毎月何時間も使ってしまう
- 退塾しそうな生徒のサインに気づくのが遅れる
- 子ども一人ひとりの学習傾向を把握しきれない
- 日々の問い合わせ対応に時間が取られる

「先生にしかできない仕事」に集中していただくため、ルーチン作業と早期察知を AI が下支えする、という設計思想で組み立てています。

---

## 主な機能

LP 上では、以下 4 つの AI 体験デモを動かせるようにしています。

### 1. 保護者レポート下書き AI
生徒の単元名・テスト点数・学習様子のメモを入力すると、保護者向けの月次レポートの下書きを生成します。
バックエンドで Anthropic Claude（Haiku 4.5）を呼び出し、過剰に煽らず温度のある文面を返すよう調整。

### 2. 退塾予兆 AI ミニデモ
架空生徒データ 3 名分（出席率・宿題提出率・テスト傾向など）を入力としてアラートレベルを判定。
「どの観点で気をつけたいか」を短文コメントで提示します。

### 3. AI 診断チャット（5 ターン制限）
塾の状況や悩みに対して、JukuMate が向いているか／どのプランがフィットしそうかを 5 ターン以内で診断します。

### 4. 個別相談予約フォーム
Cloudflare Turnstile + ハニーポット + レート制限の三重ガードで、スパム耐性を確保しつつ、
入力内容は Google Apps Script 経由で Google Sheets と Gmail に同時連携します。

## 技術スタック

| レイヤ | 採用技術 |
| --- | --- |
| Frontend | Astro v5 + Tailwind v4 + TypeScript |
| Backend (AI) | Cloudflare Workers + Hono + Anthropic Claude (Haiku 4.5) |
| Form | Google Apps Script (GAS) + Google Sheets + Gmail |
| Security | Cloudflare Turnstile / ハニーポット / レート制限 |
| Hosting | Cloudflare Pages（site）+ Cloudflare Workers（worker） |

エッジで完結する構成で、レイテンシとコストの両面で個人塾規模に現実的な設計を意識しました。

## ディレクトリ構成

```
jukumate/
├── site/        Astro 製の LP 本体（フロントエンド）
└── worker/      Cloudflare Workers 製の API（AI 呼び出し・フォーム中継）
```

`site/` は静的アセットと Astro コンポーネント、`worker/` は AI / フォーム関連のサーバサイド処理を担当します。
フロントエンドはエンドユーザーから直接見える「顔」、Worker は AI 連携やセキュリティ周りを引き受ける「裏方」、という役割分担です。

---

## ローカル起動

各ディレクトリで pnpm を使って起動できます。

```bash
# Frontend
cd site
pnpm install
pnpm dev

# Backend (Cloudflare Workers)
cd worker
pnpm install
pnpm dev
```

Worker 側は `wrangler dev` ベースで動作するため、Cloudflare アカウントを紐付けたうえで `.dev.vars` に必要な環境変数を設定してください（`.dev.vars` は公開リポには含めていません）。

---

## デプロイ

- **site/**: Cloudflare Pages にビルド出力をデプロイ
- **worker/**: Cloudflare Workers に `wrangler deploy` でデプロイ

両者を同じ Cloudflare アカウント配下でつなぐことで、`/api/*` のルーティングを Worker に渡す構成にしています。

## 公開コンテンツに関する注記

このポートフォリオ作品の性質上、いくつか注意点があります。

- LP 内に登場するメトリクス（例: 「保護者レポート作成時間を約 60% 削減」「退塾予兆検知率 約 80%・1〜2 ヶ月早く察知」「LINE 連絡下書き・月謝集計の業務時間を約 70% 削減」など）は、**架空 PF 訴求のための参考値**であり、実サービスに基づく実測値ではありません。
- 業界に存在する類似サービス（Comiru 等）への言及は公式情報の範囲にとどめており、批判的な表現や貶めるトーンは含めていません。あくまで「JukuMate がどんな立ち位置を狙っているか」を伝える文脈での参照です。
- 入力フォームや AI チャットを通じて取得される情報は、本デモでは検証目的に限定して扱う想定です。実運用時には改めて適切なプライバシーポリシー整備が必要となります。
